How to Use This Application

To see and interact with the application click on the following link: http://shiny.evl.uic.edu:3838/g2/project2/
This application visualizes data from the website https://aqs.epa.gov/aqsweb/airdata/download_files.html . You can analyze the air quality data around the US from 1980 – 2018.

To select different options such as the year, pollutant type, state and county, select from the list options on the bar on the right-hand side of the page. As you select new options the data on the screen should update to visualize the specifications you chose. Select the Daily Data option on the menu to see a bar chart of the Daily AQI for the pollutants for your chosen year, county and state. Below this you can see a bar box plot and table data for each month which shows the number of days that the AQI was good, hazardous, moderate, unknown, unhealthy, unhealthy for sensitive groups and very unhealthy.

When you select the map data section you can see the top 100 counties that have an issue with AQI or the pollutant selected based on the annual data. You can also use a slider to pick the number of counties you want to view.

The Houly Data section displays the data selected in a line graph for 2018.

Data

The data visualized in this project can be found at https://aqs.epa.gov/aqsweb/airdata/download_files.html. I downloaded all the AQI by County files. Additionally, I downloaded the Site Listing files to get the latitude and longitude locations of each county. The hourly files in 2018 for Ozone, SO2, CO, NO2, PM2.5 FRM/FEM Mass, PM10 Mass, wind and temperature were also downloaded. The Daily AQI by County for 1980-2018 were also downloaded. I used the csv files to create the visualizations you see on the application.
The data was cleaned using python script files. The hourly folder contains the 2018 files for Ozone, SO2, CO, NO2, PM2.5 FRM/FEM Mass, PM10 Mass, wind and temperature. All files except wind were reduced to the following columns:
State Code, County Code, Date Local, Time Local, Sample Measurement
A file with the state codes, county codes, state name and county name was also created so the codes could be mapped to the names and visa versa. This way spaced was saved by using integer id's rather than strings.

Getting Started

Download & Install R from https://www.r-project.org/
Download & Install RStudio from

https://www.rstudio.com/products/rstudio/download/

Open RStudio
Install the following packages by entering the following in the console:

library(shiny)
library(shinydashboard)
library(ggplot2)
library(lubridate)
library(DT)
library(jpeg)
library(grid)
library(leaflet)
library(scales)
library(reshape) #and melt function from https://www.statmethods.net/management/reshape.html
library(tidyr)
library(readr) #provided by import button for file
library(data.table)
library(dplyr)
library(raster)

Unzip files and open application To run the application press the “Run Application” button on the top right hand side
