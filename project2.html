Data

The data visualized in this project can be found at https://aqs.epa.gov/aqsweb/airdata/download_files.html. I downloaded all the AQI by County files. Additionally, I downloaded the Site Listing files to get the latitude and longitude locations of each county. The hourly files in 2018 for Ozone, SO2, CO, NO2, PM2.5 FRM/FEM Mass, PM10 Mass, wind and temperature were also downloaded. The Daily AQI by County for 1980-2018 were also downloaded. I used the csv files to create the visualizations you see on the application.

Getting Started

Download & Install R from https://www.r-project.org/
Download & Install RStudio from

https://www.rstudio.com/products/rstudio/download/

Open RStudio
Install the following packages by entering the following in the console:

library(shiny)
library(shinydashboard)
library(ggplot2)
library(lubridate)
library(DT)
library(jpeg)
library(grid)
library(leaflet)
library(scales)
library(reshape) #and melt function from https://www.statmethods.net/management/reshape.html
library(tidyr)
library(readr) #provided by import button for file
library(data.table)
library(dplyr)
library(raster)

Unzip files and open application To run the application press the “Run Application” button on the top right hand side
